========================================
신년계획 관리 웹 서비스 - 실행 가이드
========================================

[실행 순서]

1. XAMPP Control Panel 실행
   - MySQL 옆의 Start 버튼 클릭 (초록색이 되면 OK)

2. setup_database.bat 더블클릭 (최초 1회만)
   - MySQL 비밀번호 입력 (없으면 그냥 Enter)
   - "SUCCESS!" 메시지 확인

3. start.bat 더블클릭
   - 서버 시작

4. 브라우저 열기
   - 주소: http://localhost:8000

5. 로그인
   - 이메일: test@test.com
   - 비밀번호: test1234


[테스트 계정]
이메일: test@test.com
비밀번호: test1234


[문제 해결]

Q. "PHP not found" 에러
A. XAMPP 설치 경로 확인
   - start.bat 파일 열어서 17번째 줄 수정
   - set PHP_PATH=본인의XAMPP경로\php\php.exe

Q. "MySQL not found" 에러
A. XAMPP 설치 경로 확인
   - setup_database.bat 파일 열어서 9번째 줄 수정
   - set MYSQL_PATH=본인의XAMPP경로\mysql\bin

Q. MySQL이 시작 안됨
A. fix_mysql_port.bat을 관리자 권한으로 실행


[종료 방법]
- start.bat 창에서 Ctrl+C 누르기
- XAMPP에서 MySQL Stop
